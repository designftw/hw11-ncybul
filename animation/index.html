Adding Animations to Chat App

  • I added a horizontal shaking motion to the input boxes that allow users to request a new username or enter the username of the person they would like to private message which gets activated when the username is taken or when the username does not exist respectively.
    Video Demo
  • I added a transition of the background color and outline on hover to the two tabs which display the options for types of messaging (Channel-based vs. private messaging) to give users more feedback as they navigate between the two options.
    Video Demo
  • I added a bouncing motion to as well as changed the background color of the like button when it is clicked by the user in order to give them additional feedback when they have chosen to like a message.
    Video Demo
  • I added an animation which displays a helpful tooltip message when the user hovers over the button to reply to a message which gives them additional information about what will happen if they choose to reply to that message.
    Video Demo
  • I added a sliding in and out animation to the title of a thread whenever the user navigates to or from a specific thread either after creating a new thread by replying or choosing a thread that already exists from the threads page.
    Video Demo
  • When users send a message, the message input and send button briefly change color in order to give the user some feedback that their message is being sent in the chat.
    Video Demo
